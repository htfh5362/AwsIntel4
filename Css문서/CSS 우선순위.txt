CSS 우선순위 규칙 케스케이딩(Cascading)

Cascading은 작은 계단모양의 폭포를 말한다

CSS(Cascading Style Sheets)
CSS란? 

우선순위가 있는 스타일 시트라는 뜻

하나의 태그에 여러가지 방법으로 동일한 속성의 스타일을 적용하는 경우
우선순위에 따라서 적용될 스타일이 결정된다는 뜻

속성을 적용하다보면 하나의 태그에 본의 아니게 같은 속성이 겹쳐 
적용될 때가 있다 (문제점 발생)

*케스케이딩 우선순위를 결정하는 요인
1. 중요도
2. 명시도
3. 코드의 순서

*중요도
저작자 !important > 저작자 CSS > 사용자 CSS > 브라우저 CSS



1선택자에 따른 명시도
style > id > class > 태그 선택자



2같은 선택자 끼리라면 깊이(level, depth)가 깊을수록(구체적일 수록) 명시도(우선순위)가 높다
같은 종류의 선택자일 때 우선순위를 정하는 규칙


3코드 순서
선택자의 종류와 깊이가 같을 때 우선순위의 결정 방식은 코드의 순서이다
